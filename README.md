1. git clone
2. cd project
3. composer install
4. composer update
5. setting environment
6. php artisan key:generate
7. Buat database baru, kemudian buka file .env, atur username dan password MySQL kamu (default
XAMPP, username: root, password: (kosongkan)).
8. php artisan migrate
9. php artisan db:seed
10. php artisan serve
11. Buka Browser, ketik localhost:8000
