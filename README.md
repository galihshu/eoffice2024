1. git clone
2. cd project
3. composer install
4. setting environment
5. php artisan key:generate
6. Buat database baru, kemudian buka file .env, atur username dan password MySQL kamu (default
XAMPP, username: root, password: (kosongkan)).
7. php artisan migrate
8. php artisan db:seed
9. php artisan serve
10. npm install
11. npm run dev
12. Buka Browser, ketik localhost:8000
